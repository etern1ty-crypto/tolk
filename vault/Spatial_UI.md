# Spatial UI

## Основа (2026-07-15)
**Глобальная навигация = 3 вкладки** → [[Navigation_IA]]  
`Стена · Чаты · Профиль` (bottom bar).

Главная вкладка по продукту: **Чаты**.

## Внутри «Чаты»
Desktop: `[ Список | Диалог ]`  
Mobile: list → push chat; back to list.

Optional inside chat (secondary):
- **Полка** закрепов ([[Chat_Shelf]]) — кнопка / sheet, не третья «религия»  
- Профиль peer — push/sheet  

## Было (устарело как главная модель)
Горизонтальные колонны list ↔ chat ↔ wall **как единственная** IA.  
Свайп-колонны можно оставить как *жест внутри чатов*, не как root.

## Нет
- Tab-bar ×5 с Discover/shop  
- Peek-панели на 30%  

## A11y
Вкладки подписаны. Активная вкладка видна.
