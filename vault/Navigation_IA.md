# Navigation IA (основа)

> Задумка Никиты (2026-07-15) — **основа продукта**.  
> Детали UX/GPI можно шлифовать, идею не выкидывать.

## Главное
**Главная поверхность — Чаты.**  
Три вкладки (bottom nav, mobile + web mobile; desktop — та же тройка внизу или слева, но **понятная навигация**):

```
[ Стена ]   [ Чаты ]   [ Профиль ]
   лента      ★ home      я + settings
```

### Почему bottom
Знакомо. Мессенджеры держат палец внизу.  
Не «секретные» свайпы-колонны как единственный способ жить в приложении.

Desktop web: тот же смысл — **Чаты** по центру/в фокусе; Стена и Профиль — соседние вкладки (не Discover-алгоритм ради шума).

## Вкладка: Чаты (home)
- Список диалогов  
- Открытие 1:1 / группы  
- Текст, медиа, **войсы**, **кружки** (video notes)  
- Реакции (emoji; кастом — later, из профиля/настроек)  
- Echo / полка чата — secondary, не ломают home  

## Вкладка: Стена
**Лента рекомендаций постов других людей** (и своих, если «в стену»).

Источники поста:
1. **Сразу в стену** — пост задуман как публичный/полупубличный в ленту  
2. **Из профиля** — личный пост + чекбокс **«Добавить в стену»** при публикации  

Стена **подталкивает** создать пост: empty / soft CTA «Расскажите о себе».

Действия на карточке (MVP UI):
| Action | |
|---|---|
| Лайк | Must |
| Коммент | Must (thread sheet) |
| Репост **себе в профиль** | Must |
| Переслать (в чат) | Must |

Минимализм: типы постов (фото / текст / «из профиля») отличаются тонко, без зоопарка chrome.

**Не** алгоритм Discover «для вовлечённости любой ценой».  
Рекомендации = простой ranking / recency / friends first (protocol later).

## Вкладка: Профиль
- Оформление: аватар, имя, био, **фон / banner**, кастомизация  
- **Посты** на своей странице  
- При публикации: **«Добавить в стену»**  
- **Настройки** живут здесь (или вход из профиля)  
- Чужой профиль: открывается из стены / чата → «Написать», его стена/посты  

## Кружки и войсы (чат)
| | |
|---|---|
| Войс | Hold / tap record, waveform, playback — как ожидаешь |
| Кружок | Круглое video note, запись «сейчас» |
| Камера | Front/back switch |
| Вспышка | Toggle (rear) |
| Эффекты | Blur / fun (котики…) — **глубоко спрятаны** (long path, advanced) |
