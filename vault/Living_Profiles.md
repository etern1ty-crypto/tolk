# Living Profiles

Третья вкладка [[Navigation_IA]] + карточка другого человека.

## Свой профиль (вкладка)
| | |
|---|---|
| Аватар, имя, @username, био | Must |
| **Кастомизация** фона / banner | Must (UI mock ok) |
| Лента **своих** постов | Must |
| Композер + **«Добавить в стену»** | Must |
| Вход в **Настройки** | Must |
| Свои emoji / packs (later) | Should |

Профиль должен ощущаться **красивым**, не «формой из госуслуг». Минимализм ≠ серость.

## Чужой профиль
- Из стены (автор) / из чата  
- **Написать**  
- Его посты  
- Без редактирования  

## Настройки
Живут **в профиле** (или из шапки профиля).  
Не отдельная 4-я вкладка.

## Связь со стеной
Пост профиля с `on_wall` попадает в [[User_Wall|Стену]] другим.
