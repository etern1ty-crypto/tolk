# Стена (лента)

> Основа (Никита, 2026-07-15): стена — **подборка / рекомендации постов** других людей.  
> Навигация: первая вкладка [[Navigation_IA]].

## Что это
Лента, где встречаются:
- посты, созданные **сразу в стену**;
- **личные** посты из профиля, если при публикации стояло **«Добавить в стену»**.

Тонко отличаются (бейдж «профиль» / «стена»), без тяжёлого chrome.

## Зачем
1. Живой soft-onboarding: «расскажите о себе»  
2. Увидеть людей → зайти в профиль → **Написать**  
3. Не превращать мессенджер в Instagram Explore  

## Действия карточки
- **Лайк**  
- **Комментарий** (sheet / thread)  
- **Репост в свой профиль** — пост появляется у тебя  
- **Переслать** в чат  

Без публичных vanity-счётчиков в лицо, если мешает минимуму — показывать спокойно (число лайков secondary).

## Создание
| Где | |

Не алгоритм « sorберём 4 часа».  
Не stories.  
Не полка закрепов чата ([[Chat_Shelf]] — другое).
