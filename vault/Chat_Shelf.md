# Полка чата (Chat Shelf)

> Раньше ошибочно называлась «стеной». **Стена** теперь — лента профиля ([[User_Wall]]).  
> **Полка** — полка важных сообщений и медиа *этого* диалога.

## Боль
Фото, файлы, «это важно» тонут в истории через день.  
В TG — скролл или «в избранное». Отдельный канал — лишняя сущность.

## Решение
**Полка** — полка закрепов и медиа **этого** чата (1:1 или группа).  
Не глобальная лента. Не stories. Не [[User_Wall|стена профиля]].

## Как попадает
1. Long-press сообщения/медиа → **На полку**  
2. Опционально при отправке: «и на полку»  
3. В группе: права (все / admin) — см. [[Open_Questions]]

## Что лежит
Фото, видео, файлы, короткие текстовые закрепы, ссылки с превью.  
Карточка → [[Structure_of_a_Post]] (режим `shelf`).

## Навигация
Свайп или кнопка «Полка» ([[Spatial_UI]], [[Gesture_Mapping]]).  
С полки: **В чат** → jump к исходному сообщению + краткая подсветка.

## Empty state
Одна строка: «Долгое нажатие на сообщение → На полку». Без баннерного ада.

## Порядок
MVP: новые сверху. Drag-reorder — Could.  
Конфликт: LWW по server time.

## Снять с полки
Снять ≠ удалить из чата. Рвётся только `ShelfItem` ([[Data_Graph]]).

## Анти-клон
Нет алгоритма, лайков, публичной «стены для всех».

## MVP
Must: pin на полку, лента полки, jump-to-chat.  
Could: drag, сложные роли.

Связь: pin из [[Messages_Basics]], вход из [[Chat_Settings]] / колонны [[Spatial_UI]].  
Не путать со [[User_Wall|стеной пользователя]].
