# Contributing

## Где код

- UI: `apps/web`
- Продукт: `vault/` (Obsidian)

## Правила

1. Не выдумывать фичи мимо `vault/MVP.md` и `vault/Navigation_IA.md`.  
2. UI: monochrome, lucide, hairline feeds.  
3. Перед PR: `npm run build` из `apps/web`.

## Commits

Коротко, по-русски или en: `feat(web): …`, `docs(vault): …`.
