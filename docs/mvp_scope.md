# MVP Scope & Must UI Screens (Tolk Frontend)

## Ticket Table: Must UI Screens (v1 Frontend)

| Category | Screen / Component | Description / Requirements | Priority |
| :--- | :--- | :--- | :--- |
| **Auth & Account** | Login Screen | Phone + OTP entry. Simple, direct. | Must |
| **Auth & Account** | Profile Setup | Name, Avatar setup after OTP. | Must |
| **Core Base** | Chat List | Main view. Rows of chats. Search entry point. | Must |
| **Core Base** | Global Search | Find chats / messages. | Must |
| **Core Base** | Chat View (1:1 & Group) | Messages, input field, attach button, voice record. | Must |
| **Core Base** | Media Picker & Attach | Sheet/overlay to pick photos/files. | Must |
| **Core Base** | Chat Settings | Group info, mute, members, leave group. | Must |
| **Settings** | Settings Hub | Account, Notifications, Appearance, Privacy, Storage. | Must |
| **Differentiators** | The Chat Wall (Стена) | Dedicated column/screen for pinned media & messages in a chat. | Must |
| **Differentiators** | Echoes UI | "Echo" send mode toggle, quiet push indication, Echo sheet/chip. | Must |

## Excluded from MVP (Won't / v2+)
- Algorithmic feeds / Discover
- Stories
- Mini-apps / stores
- Ads in chats
- Heavy animations / 5-tab bar
- Live Thoughts (Could/v2)
- Whispers / Ephemeral voice (Could/v2)
- E2EE 1:1 (v2)
